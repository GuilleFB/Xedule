About This Application

This application allows users to schedule and automatically publish tweets to X (formerly Twitter) using the official X API. Built with Django and Celery, the platform handles tweet scheduling, posting, and error management in a fully automated and reliable way.

  • 📅 Tweet Scheduling: Compose tweets and choose when to post them.
  • ⚙️ Automated Publishing: Tweets are published automatically at the scheduled time.
  • 🔁 Retry Logic: If posting fails, the system retries with exponential backoff.
  • 📊 Status Tracking: See which tweets are pending, published, or failed.
  • 🔒 Secure Authentication: OAuth credentials are used to ensure secure access to the X API.

This tool is ideal for individuals, content creators, and businesses who want to automate their social media workflow and ensure timely and consistent communication on the X platform.
